# GoLang-Basics

First, install GoLang and do the basic setup

[Getting Started](https://golang.org/doc/install)

## Instructions

Run `main.go` file:

`go run main.go`

Now the server is running and you can do the `GET` requests
* localhost:3000/ 
  * Content-Type
    * text/plain
    * text/html
    * application/xml
    * application/json
    * image/gif
* localhost:3000/test
